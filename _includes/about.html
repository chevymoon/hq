The Nika Database Service Contract is primarily built as an example but it can serve as a private database for many features; online games, agricultural data, insurance records, weather reports and mostly anything not hiding sensitive data. Although the contract is 100% private on the blockchain and does not expose information, it still is not secure from outside elements outside the blockchain, such as someone taking your ethereum wallet.



1. Make sure you have metamask installed or any other tool to work with the ethereum blockchain

2. You need to have some gas (ether) in your wallet and at least one Nika token

3. You need to approve the contract to send one Nika token from your wallet. To do this, you need to load the Nika token contract.

The previous link will take you to the Dapp page, you need to click the Web3 link in the top right page and activate Web3. If you're connected to metamask and loggin in, a window will pop-up and you have to approve that.

4. Click the Write Contract button, the click the Approve function.

Copy the Database contract address: 0x519e5585540ac8d5770cfcd8175bbbd2a6430367

Paste it into the spender field. Then since Nika Token has five decimal places, put 100000 for one Nika Token or if you want more slots, put more. Don't worry, the database contract will only charge you for one slot at a time so it's impossible to overspend here.



You should see this window once click the write button. Verify the transaction.



4. Good, now you're ready to roll. Visit the Database Contract Dapp page. Again make sure you're connect to Web3 and have Metamask running.
Click Create Database button and hit write, approve the transaction. Congratulations! You now have a database on Ethereum!
*You need at least one Nika token for it to transfer and get set up*

5. You can edit two strings at a time, 1 & 2, 3 & 4, 5 & 6, 7 & 8. You can also edit the address from this menu as well as the two numbers. You can navigate between the slots by selecting choose your slot.

6. Unfortunately, you can't really read information through that Dapp page. You can head over to remix, paste the code there and as long as you use the same metamask wallet, you'll be able to read information. Only your wallet can read information, no one else.
